Node.js ga Kirish

Node.js nima?

Node.js — bu Chrome’ning V8 JavaScript dvigateliga asoslangan JavaScript ishlash muhiti.

Node.js ni o‘rnatish

Node.js rasmiy sayti dan yuklab oling va o‘rnating.

NPM dan foydalanish

NPM (Node Package Manager) yordamida paketlarni boshqarish mumkin: npm install package-nomi

Asosiy modullar

Node.js quyidagi ichki modullarga ega: fs, http, path.

Express.js ga kirish

Express.js — bu Node.js da veb-ilovalar yaratish uchun keng tarqalgan framework.
